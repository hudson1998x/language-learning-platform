# LLE Language Learning Platform

**Learn languages through real experiences, not vocabulary lists.**

LLE is a free and open-source language acquisition platform designed around a simple belief:

> People learn languages by using them.

Instead of memorising isolated words and grammar rules, learners engage with music, conversations, social interactions, and realistic scenarios that mirror how language is used in everyday life.

Every phrase can be understood, practised, and retained through intelligent spaced repetition.

---

# Open Source & Forever Free

LLE is open source and will remain free forever.

Language learning should be accessible to everyone, regardless of income, location, or background.

The project is built around a commitment to:

* Free access for all learners
* Open source development
* Community-driven improvement
* Transparency and ownership
* No subscriptions
* No locked features
* No artificial usage limits

Anyone can inspect the code, contribute improvements, report issues, or help shape the future of the platform.

---

# Philosophy

Most language-learning platforms follow this pattern:

```text
Vocabulary → Grammar → Exercises → Real Usage
```

LLE reverses the process:

```text
Real Usage → Understanding → Practice → Retention
```

The goal is not simply to teach vocabulary.

The goal is to build fluency.

Learners should spend less time studying language and more time experiencing it.

# Current Features

## 🎵 Music Translation

Learn through songs and lyrics from the music you already enjoy.

For every lyric, LLE provides:

* Original text
* English translation
* Easy-to-read pronunciation
* Cultural and idiomatic explanations
* Context-aware meaning

### Example

**Original**

```text
Me gusta esta canción
```

**Pronunciation**

```text
may goo-stah es-tah can-see-on
```

**Translation**

```text
I like this song
```

**Cultural Context**

```text
A common expression used when discussing music preferences in everyday Spanish conversation.
```

Rather than providing a literal translation alone, LLE helps learners understand how native speakers actually use language.

---

## 📚 Spaced Repetition Flash Cards

LLE includes a built-in flash card system powered by Spaced Repetition Scheduling (SRS).

Cards can contain:

* Original phrase
* Translation
* Pronunciation
* Cultural notes
* Additional learning context

Every card is designed to reinforce language encountered through meaningful experiences.

---

## ⚡ Instant Flash Card Creation

Any translated lyric can become a flash card in seconds.

Workflow:

1. Highlight a lyric or phrase
2. Click **Create Flash Card**

LLE automatically captures:

* Original text
* Translation
* Pronunciation
* Cultural notes

No copy-pasting.

No duplicate work.

Simply learn something useful and save it for later review.

## 💬 LeMessage

LeMessage is a messaging platform designed specifically for language learners.

Unlike traditional messaging applications, users do not message real people.

Instead, they interact with AI-powered personalities designed to simulate natural conversations.

Learners can:

* Practice everyday conversations
* Build confidence
* Learn conversational vocabulary
* Develop fluency through repetition
* Explore different personalities and speaking styles

The objective is not chatting for entertainment.

The objective is conversational fluency.

---

## 📰 Leddit

Leddit is a Reddit-inspired language practice environment.

Despite the name, it is not a social network.

It is a safe space where learners can practise participating in online communities before doing so in the real world.

Users can:

* Read discussion threads
* Write comments
* Participate in debates
* Learn internet slang
* Understand informal writing styles
* Develop confidence expressing opinions

AI-generated participants create realistic discussions while removing the anxiety and pressure often associated with posting publicly.

Think of it as a training ground for real online communication.
